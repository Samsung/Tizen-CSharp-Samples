# IMESample
This is a simple project for C# IME(Input Method Editor) Application, which lets you create your own IME.
SampleIME is mainly designed for Tizen Mobile profile.

![portrait_ime](./ScreenShots/portrait_ime.png)


### Verified Version
* Xamarin.Forms : 4.5.0
* Tizen.NET : 6.0.0
* Tizen.NET.SDK : 1.0.9


### Supported Profile
* Mobile
