# Sensors

The Sensors sample application demonstrates how you can create an application using the following APIs:

* Xamarin.Forms
* Tizen.Wearable.CircularUI.Forms
* Tizen FX APIs (Tizen.Sensor, Tizen.Security)
* SkiaSharp
